## Options

<snippet id="snippet-options-all">

<snippet id="snippet-options">

`-r`, `--retry`
: Retries downloading when it fails, with optional number of times to retry (Defaults to 2)

</snippet>

`-h`, `--help`
: Show the help message and exit

</snippet>
